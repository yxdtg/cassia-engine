# Cassia Engine

一个专注于 2D、易用性和高性能的游戏引擎。

# 文档

# [cassia-engine-docs](https://yxdtg.github.io/cassia-engine-docs/zh/)

# 使用 create-cassia-engine 创建项目

# [create-cassia-engine](https://www.npmjs.com/package/create-cassia-engine)

## 引擎构建

### 开发环境

```bash
npm run dev
```

### 生产环境

```bash
npm run build
```
